Random number generation

Random integer

To get a random number, we first need to use the predefined function, randomize(), along with another function specific to the type of random number we want. 

randomize()
var my_random_int = randi()
print(my_random_int)

Random integer within specified range


var rand_gen = RandomNumberGenerator.new()

func _ready():
	rand_gen.randomize()
	var my_random_int = rand_gen.randi_range(45,75)
	print(my_random_int)

Random float

The randf() function generates a random float between 0 and 1.


func _ready():
	randomize()
	var my_random_float = randf()
	print(my_random_float)

Random float within specified range


var rand_gen = RandomNumberGenerator.new()

func _ready():
	rand_gen.randomize()
	var my_random_float = rand_gen.randf_range(45.5,75.34)
	print(my_random_float)
