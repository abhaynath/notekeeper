Graphviz basics

Installation

  1. Download installer from Graphviz download link
  2. Install it for all users
  3. Make sure that 
    C:\Program Files\Graphviz\bin
    path is added in system variables.
  4. To check the installation is successfull, run following command. 
    dot --help

Hello World

  1. Create a new file g.dot
  2. Add following content in the file. 
    
    digraph G {
        Hello -> World;
    }
  3. To generate svg file from the source file, run following command. 
    dot -Tsvg g.dot -o helloworld.svg
