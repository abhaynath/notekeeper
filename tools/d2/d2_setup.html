Setup

  1. Download the installer from following url 
    https://github.com/terrastruct/d2/releases
  2. To check the installation, 
    d2 --version
  3. Create a new file hello.d2 will following code. 
    x -> y: hello world
  4. To export it as SVG, 
    d2 hello.d2 myhello.svg
  5. To export as png 
    d2 hello.d2 myhello.png
