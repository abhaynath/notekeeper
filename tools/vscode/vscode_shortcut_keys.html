Adding custom key bindings for switching between Editor/Terminal

You can use following key combinations for Editor/Terminal switching

    {
        "key": "alt-`",
        "command": "workbench.action.focusActiveEditorGroup",
        "when": "terminalFocus"
    },
    {
        "key": "alt+`",
        "command": "workbench.action.terminal.focus",
        "when": "!terminalFocus"
    }
