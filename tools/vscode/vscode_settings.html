VS Code

Custom Extension Path

  1. Create an empty folder. For example, 
    F:\Programs\VsCodeExtensions\extensions
  2. "C:\Program Files\Microsoft VS Code\Code.exe" --extensions-dir="F:\Programs\VsCodeExtensions\extensions"
  3. Update 'target' text box with following path

Update Context menu for 'Open With...'

  1. Open Regedit as Administrator
  2. Navigate to Computer\HKEY_CLASSES_ROOT\directory\shell\VSCode\command
  3. Append --extensions-folder=[alternative path for extensions folder] the value (Default)
  4. Navigate to Computer\HKEY_CLASSES_ROOT\directory\background\shell\VSCode\command
  5. Append --extensions-folder=[alternative path for extensions folder] the value (Default)
