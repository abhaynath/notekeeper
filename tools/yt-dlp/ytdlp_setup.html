yt-dlp Index

yt-dlp setup

  1. Download installer from yt-dlp download link
  2. Install it for all users
  3. Make sure that 
    F:\Programs\yt-dlp\
    path is added in system variables.
  4. To check the installation is successfull, run following command. 
    yt-dlp --help

ffmpeg setup

  1. Download installer from ffmpeg download link
  2. Install it for all users
  3. Make sure that 
    F:\Programs\ffmpeg\
    path is added in system variables.
  4. To check the installation is successfull, run following command. 
    ffmpeg -version
