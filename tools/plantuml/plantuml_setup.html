Setup

  1. Download the jar file from following url PlantUTML download page
  2. Rename the jar file to plantuml
  3. Create a new folder and copy the jar file to that folder
  4. Add this folder to Environment Variable
  5. Create a new batch file puml.bat in the same directory. Add following code inside the file.

    puml.bat

    java -jar F:\Programs\PlantUML\plantuml.jar %1 %2 %3 %4 %5 %6 %7
  6. Make sure the folder is Environment Path
  8. To check the installation run following command, 
    puml -version
    This should produce output like This

Simple State diagram

  1. Create a new text file state.puml
  2. Add following content in the File 
    
@startuml

[*] --> State1
State1 --> [*]
State1 : this is a string
State1 : this is another string

State1 -> State2
State2 --> [*]

@enduml
  3. To generate png, 
    puml state.puml
  4. To generate png in specific folder, 
    puml -o output_folder state.puml
  5. To generate svg, 
    puml state.puml -tsvg
    To Generate svg in specific folder 
    puml -o output_folder state.puml -tsvg
