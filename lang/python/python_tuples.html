Tuples

Tuples are used to store multiple items in a single variable.

Tuple is one of 4 built-in data types in Python used to store collections of data, the other 3 are List, Set, and Dictionary, all with different qualities and usage.

A tuple is a collection which is ordered and unchangeable.

Tuples are written with round brackets.


# Tuple with all elements of same type
fruits = ("Apple", "Orange", "Banana")

# Tuple with mixed data types
computer = ("Dell", 30000, 512, "1920x1080")

# Print Tuple items
print(fruits)
print(computer)

# Access individual Tuple item
print(fruits[0])                    # Apple
print(computer[3])                  # 1920x1080

# Iterate all items in tuple
for f in fruits:
    print(f)

# Iterate range in tuples
for c in computer[1:3]:
    print(c)

# get tuple type
print(type(fruits))

# get length of tuple
print(len(fruits))
print(len(computer))

# Check if item exists
if "Dell" in computer:
    print("Dell is present")
else:
    print("Not found")

Change Tuple values

Once a tuple is created, you cannot change its values. Tuples are unchangeable, or immutable as it also is called.

But there is a workaround. You can convert the tuple into a list, change the list, and convert the list back into a tuple.


player = ("Rohit Sharma", "India", "Mumbai Indians")
print(player)           # ('Rohit Sharma', 'India', 'Mumbai Indians')

temp = list(player)
temp[0] = "Virat Kohli"

player = tuple(temp)
print(player)           # ('Virat Kohli', 'India', 'Mumbai Indians')

Add Item

Since tuples are immutable, they do not have a build-in append() method, but there are other ways to add items to a tuple.

Convert into a list: Just like the workaround for changing a tuple, you can convert it into a list, add your item(s), and convert it back into a tuple. 


india = ("Rohit", "Shikhar", "Virat")
temp = list(india)
temp.append("Hardik")
india = tuple(temp)
print(india)                # ('Rohit', 'Shikhar', 'Virat', 'Hardik')

Add tuple to a tuple: You are allowed to add tuples to tuples, so if you want to add one item, (or many), create a new tuple with the item(s), and add it to the existing tuple 


india = ("Rohit", "Shikhar", "Virat")
temp = ("Hardik",)
india += temp
print(india)                            # ('Rohit', 'Shikhar', 'Virat', 'Hardik')

Remove items from Tuple

Items in Tuple cannot be removed directly.

Tuples are unchangeable, so you cannot remove items from it, but you can use the same workaround as we used for changing and adding tuple items


india = ("Rohit", "Shikhar", "Virat")
temp = list(india)
temp.remove("Shikhar")
india = tuple(temp)                     # ('Rohit', 'Virat')

Unpacking Tuple

When we create a tuple, we normally assign values to it. This is called "packing" a tuple.

We are also allowed to extract the values back into variables. This is called "unpacking".


india = ("Rohit", "Shikhar", "Virat")
(hitman, gabbar, chiku) = india

print(hitman, gabbar, chiku)            # Rohit Shikhar Virat

Using Asterisk*

If the number of variables is less than the number of values, you can add an * to the variable name and the values will be assigned to the variable as a list 


fruits = ("apple", "banana", "cherry", "strawberry", "raspberry")

(green, yellow, *red) = fruits

print(green)                        # apple
print(yellow)                       # banana
print(red)                          # ['cherry', 'strawberry', 'raspberry']


(first, *middle, last) = fruits
print(first)                        # apple
print(middle)                       # ['banana', 'cherry', 'strawberry']
print(last)                         # raspberry

Join Tuples

To join two or more tuples you can use the + operator


tuple1 = ("a", "b", "c")
tuple2 = (1, 2, 3)

tuple3 = tuple1 + tuple2
print(tuple3)
