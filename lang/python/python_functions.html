Python

Functions

A function is a block of code which only runs when it is called.

You can pass data, known as parameters, into a function.

A function can return data as a result.


def get_full_name(first_name, m_name, last_name):
    full_name = first_name + " " + m_name[0] + ". " + last_name
    return full_name


print(get_full_name("Abhay", "Kiran", "Nath"))

# Abhay K. Nath

Arbitrary Arguments, *args

If you do not know how many arguments that will be passed into your function, add a * before the parameter name in the function definition.

This way the function will receive a tuple of arguments, and can access the items accordingly:


def get_sum(*nums):
    sum = 0
    for n in nums:
        sum += n
    return sum


total = get_sum(10, 35, 66, 21, 52, 31)
print(total)

Keyword Arguments

You can also send arguments with the key = value syntax. This way the order of the arguments does not matter.


def get_user(f_name, m_name, l_name):
    user = {"first_name": f_name, "middle_name": m_name, "last_name": l_name}
    return user


u1 = get_user("Abhay", "Kiran", "Nath")
u2 = get_user(m_name="Ramesh", f_name="Sachin", l_name="Tendulkar")

print(u1)
print(u2)

# {'first_name': 'Abhay', 'middle_name': 'Kiran', 'last_name': 'Nath'}
# {'first_name': 'Sachin', 'middle_name': 'Ramesh', 'last_name': 'Tendulkar'}

Arbitrary Keyword Arguments, **kwargs

If you do not know how many keyword arguments that will be passed into your function, add two asterisk: ** before the parameter name in the function definition.

This way the function will receive a dictionary of arguments, and can access the items accordingly:


def get_user(**data):
  print(data["email"])


get_user(first_name="Abhay", email="abhaynath@gmail.com")

Default Values

If we call the function without argument, it uses the default value


def get_data(x, num=5):
  return x * num


a = get_data(5)
b = get_data(35, 10)

Recursion

Python also accepts function recursion, which means a defined function can call itself.

Recursion is a common mathematical and programming concept. It means that a function calls itself. This has the benefit of meaning that you can loop through data to reach a result. 


def factorial(num):
    if num == 1:
        return num
    return num * factorial(num-1)


print(factorial(5))
