Create virtual environment

Virtual environment allows us to maintain different versions of packages in each individual projects.

  1. Create a new folder. Name it as env-test
  2. Create another new folder named .venv in env-test folder.
  3. Create a file .env This will contain important key-value pair data.
  4. To create a virual environment automatically, we need to install a package using pipenv
  5. Suppose, you want requests package in your project. You need to run following command from the env-test directory. 
    pipenv install requests
  6. With this command, virtual environment will be generated. Pipfile and Pipfile.lock will be generated automatically.
Virtual environment actions

  • Activate environment

    When virual environment is activated, it will use local python installation

    pipenv shell

  • Deactivate environment

    exit

  • Create a virtual enviornment with specific python version

    pipenv --python 3.6

  • Remove virtual enviornment

    pipenv --rm

  • Recreate virtual enviornment from the pipfile

    pipenv install

  • Get virual enviornment location

    pipenv --venv

  • Checking virtual enviornment status

    pipenv check

  • Show dependency graph

    pipenv graph

  • Create the lock file

    pipenv lock

  • Install the packages in lock file

    pipenv install --ignore-pipfile

        Using enviornment variables
        

          

          
    
            
  • 
              Create .env file in the root project folder. Add required key-value pairs.
              
    SECRET_KEY = ABHAYS SECRET PASSWORD
    
            
    • 
            
  • 
              Create main.py file in the root folder. Add following snippet which accesses enviornment variables.
              
    
import os

passkey = os.environ['SECRET_KEY']

print(f"My Passkey is {passkey}")

        Running custom commands
        

          
    
            
  • 
              add [scripts] tag in pipfile like this

              
    [scripts]
buildexe = "pyinstaller --onefile test.py"

  • 
              To run buildexe command from virtual enviornment

              
    pipenv run buildexe
