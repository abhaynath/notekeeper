Flags in file operations

Flags

The key function for working with files in Python is the open() function.

The open() function takes two parameters; filename, and mode.

There are four different methods (modes) for opening a file

Flag Mode Details
r Read Default value. Opens a file for reading, error if the file does not exist
a Append Opens a file for appending, creates the file if it does not exist
w Write Opens a file for writing, creates the file if it does not exist
x Create Creates the specified file, returns an error if the file exists

In addition you can specify if the file should be handled as binary or text mode

Flag Mode Details
t Text Default value. Text mode
b Binary Binary mode (e.g. images)
Reading files

Reading file

The open() function returns a file object, which has a read() method for reading the content of the file


  f = open("F:\\test.txt", "r")
  print(f.read())
  f.close()

Read only part of file

By default the read() method returns the whole text, but you can also specify how many characters you want to return


  f = open("F:\\test.txt", "r")
  print(f.read(5))
  f.close()

Reading lines

You can return one line by using the readline() method


  f = open("F:\\test.txt", "r")
  print(f.readline())
  f.close()

Read the file line by line

By looping through the lines of the file, you can read the whole file, line by line


f = open("F:\\test.txt", "r")
for line in f:
print(line)

f.close()
Writing to file

Write to an Existing File

To write to an existing file, you must add a parameter to the open() function

Flag Mode Details
a Append will append to the end of the file
w Write will overwrite any existing content

Following code appends timestamps in the file.


  import datetime
  
  current_time = datetime.datetime.now()
  timestamp = str(current_time)
  
  f = open("F:\\test.txt", "a")
  f.write(timestamp+"\n")
  f.close()
Write and Read a JSON Configuration file

Create new file

To create a new file in Python, use the open() method, with one of the following parameters

Flag Mode Details
x Create will create a file, returns an error if the file exist
a Append will create a file if the specified file does not exist
w Write will create a file if the specified file does not exist

Following program creates the python object, writes it to the json file and then it reads it again.


  import json
  import datetime
  
  
  config = {
      'userId': "A101G",
      'password': "abc123",
      'ip': '172.31.16.71',
      'max_connections': 1000,
      'domain': 'software.com',
      'isActive': True,
      'users_list': ["abhay", "sachin", "nazma"],
      'server_details': None
  }
  
  # get current timestamp
  current_time = datetime.datetime.now()
  timestamp = str(current_time)
  
  # set last modified timestamp
  config['last_modified'] = timestamp
  print(config)
  
  # convert object to json string
  strJson = json.dumps(config)
  print(strJson)
  
  # writes the json string to file
  filePath = "F:\\config.json"
  f = open(filePath, "w")
  f.write(strJson)
  f.close()
  
  
  # read the contents of the file
  f = open(filePath, "r")
  strConfig = f.read()
  f.close()
  
  # convert the string json to object
  newConfig = json.loads(strConfig)
  print(newConfig)
Delete file

Delete File

To delete a file, you must import the OS module, and run its os.remove() function


import os

filePath = "F:\\test.txt"
if os.path.exists(filePath):
os.remove(filePath)
print("File deleted")
else:
print("The file does not exist")

Rename single file

Rename single file

Python comes with a built-in function, rename(), as part of the os library. As the name implies, the function is used to rename a file. Before diving into how practical examples, let’s take a look at what makes up the function, including its various parameters: 


  # Understanding the Python os.rename() Function
  os.rename(
     src, 
     dst, 
     *, 
     src_dir_fd=None, 
     dst_dir_fd=None
  )
Parameter Description Default Argument Accepted Values
src The path for the file that you want to rename string or path-like
dst The destination path for the file that you want to use string or path-like
src_dir_fd The source file directory None string or path-like
dst_dir_fd The destination file directory None string or path-like 

  import os
  
  oldFileName = 'my_file.txt'
  newFileName = "new_file_150.py"
  try:
      os.rename(oldFileName, newFileName)
  except FileNotFoundError:
      print(f"{oldFileName} not found")
  except FileExistsError:
      print(f"{newFileName} already exists.")
  else:
      print("File renamed successfully.")
Rename multiple files

Rename multiple files

To rename multiple files using Python, you can combine the os.rename() function with a Python for loop. A for loop lets you iterate over a list of items, such as a list of file names. 


import os

dir = 'files/'

for file in os.listdir(dir):
old_filepath = os.path.join(dir, file)
new_name = 'old_' + file
new_filepath = os.path.join(dir, new_name)

os.rename(old_filepath, new_filepath)
Copy file

Copy file to particular path with filename

We can file to copy file to a particular location with specific file name.


import shutil

sourceFile = 'rename.py'
destinationFile = 'files/newRename.py'

shutil.copyfile(sourceFile, destinationFile)

Copy file to particular folder

We can directly copy to particular folder (is should exists) without specifying file.


import shutil

sourceFile = 'rename.py'
destination_folder = 'files2/'

shutil.copy(sourceFile, destination_folder)

Copy a File as a File Object

You can also copy a file as a file object by using the shutil.copyfileobj() method. This method, instead of taking file paths, takes file-like objects as its arguments. 


import shutil

source_file_path = 'rename.py'
source_file = open(source_file_path, 'rb')

destination_file_path = 'files3/rename444.py'
destination_file = open(destination_file_path, 'wb')

shutil.copyfileobj(source_file, destination_file)
