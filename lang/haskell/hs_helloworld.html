Simple hello world application in Haskell

  1. Create an empty directory 'helloworld'
  2. Create a new go file 'helloworld.hs'
  3. Add following code snippet in 'helloworld.hs' 
    main = putStrLn "Hello World"
  4. To build the project 
    ghc -o hello helloworld.hs
    This command will create 'hello.exe' in the project directory.
  5. To run the application, 
    .\hello
