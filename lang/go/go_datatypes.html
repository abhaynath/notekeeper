Simple hello world application in Go

  1. Create an empty directory 'helloworld'
  2. Create a new go file 'hello.go'
  3. Add following code snippet in 'hello.go' 
    
    package main

    import "fmt"
    
    func main() {
        fmt.Println("Hello, world !")
    }
  4. To build the project 
    c:\project\helloworld\go build
    This command will create 'helloworld.exe' in the project directory.
  5. To run the application, 
    c:\project\helloworld\helloworld
  6. To build and run project in one command, 
    c:\project\helloworld\go run hello.go
