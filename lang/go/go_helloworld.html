Simple hello world application in Go

  1. Create an empty directory 'helloworld'
  2. Run following command to initialize module 
    go mod init github.com/abhaynath/helloworld
  3. Create a new go file 'main.go'
  4. Add following code snippet in 'main.go' 
    
    package main

    import "fmt"
    
    func main() {
        fmt.Println("Hello, world !")
    }
  5. To build the project 
    c:\project\helloworld\go build
    This command will create 'helloworld.exe' in the project directory.
  6. To run the application, 
    c:\project\helloworld\helloworld
  7. To build and run project in one command, 
    c:\project\helloworld\go run .
