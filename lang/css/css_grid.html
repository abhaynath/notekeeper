Flex Parent Properties

Property Values Description
display flex / inline-flex Converts element into flex container
flex-direction row(default) / row-reverse / column / column-reverse Sets the direction of flex-items
flex-wrap nowrap (default) / wrap / wrap-reverse Wrapping when available space
flex-flow flex-direaction flex-wrap Combine flex-direction and flex-wrap in single property. e.g. ( flex-flow : row wrap; )
justify-content center/ flex-start / flex-end / space-between / space-evenly / space-around justifies the content
align-items stretch / flex-start / flex-end (container height is mandatory to work align-items. It aligns the item vertically)
align-content stretch|center|flex-start|flex-end|space-between|space-around|initial|inherit; container height and flex-wrap:wrap is mandatory. It aligns vertically

Flex-item property

Property Values Description
order 0/1/... set the order in which element is displayed. default value is 0. Elements are displayed from lower to higher order number.
flex-grow 0/1/... It specifys the size of the element in available container area. By default value is 0 (elment is size is not changed) flex-grow:1; (All flex-items are resized evenly) flex-grow: 5; (element grows five times than other elements)
flex-shrink similar to flex-grow properties Its opposite of flex-grow. It shrinks the element width
flex-basis 100px / 40% Sets the initial width of the element. accepted values are in pixels, percentage etc. NEVER use width property in flex. Use flex-basis instead. flex-grow and flex-shrink will be applied on top of flex-basis value.
flex flex: 1 0 200px; It is combination of three properties (flex-grow, flex-shrink & flex-basis). We can specify all three properties in one go with flex
align-self flex-start / flex-end / center / auto / stretch This property individually aligns the element in container.
