Using CSS Variables

Declaring root scope variables

Use -- before the variable name to declare css variable. Variables declared in root scope can be used anywhere in page. 


    :root 
    {
    --main-bg-color: gray;
    --main-txt-color: white;
    --header-bg-color: black;
    }

Declaring class scope variables

Here we are declaring few class scoped varibles which are available only in .box class. 


    .box 
    {
        --box-bg: #fff;
        --box-theme-color: rgb(112, 3, 139);
        --box-padding: 10px;
    }

    .box 
    {
        border: 3px solid var(--box-theme-color);
        background-color: var(--box-bg);
        color: var(--box-theme-color);
        padding: var(--box-padding);
        box-shadow: 10px 5px 5px var(--box-theme-color);
    }

Using JavaScript to get and set css variables

Header div is using background-color from root scope.

    #myheader 
    {
        background-color: var(--header-bg-color);
        color: white;
        padding: 5px;
        text-align: center;
    }
        
    <div id="myheader">Header with default black background</div>

    <h1 id="txtcolor"></h1>
        
    const box = document.querySelector(".box");
    const boxStyles = getComputedStyle(box);
    const boxThemeColor = boxStyles.getPropertyValue("--box-theme-color");
    document.getElementById("txtcolor").innerHTML = boxThemeColor;

    const header = document.querySelector("#myheader");
    header.style.setProperty("--header-bg-color", boxThemeColor);
