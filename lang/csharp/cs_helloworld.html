Console Output Formatting

  1. Display value and string (Using Substitution markers)

    
    string name = "Sachin Tendulkar";
    int score = 200;

    Console.WriteLine("Player {0} scored {1} runs.",name,score);
    This will produce following Output 
        Player Sachin Tendulkar scored 200 runs.

  2. Display value and string (Using String interpolation)

    
    string name = "Mahendra singh Dhoni";
    int score = 183;

    Console.WriteLine($"Player {name} scored {score} runs.");
    This will produce following Output 
        Player Mahendra singh Dhoni scored 183 runs.

  3. Formatting decimals to specific decimal places

    
    double myDouble = 12.345678;

    Console.WriteLine("{0,-10:F4} -- Fixed Point, Four decimal places",myDouble);
    Console.WriteLine("{0,-10:E3} -- Scientific Notation, 3 decimal places",myDouble);
    This will produce following output 
        12.3457 --Fixed Point, Four decimal places
    1.235E+001 -- Scientific Notation, 3 decimal places
