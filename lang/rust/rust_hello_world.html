Rust

Hello World

  1. Run cargo command to create new project hello-world 
    cargo new hello-world
  2. In src/main.rs file, add following code. 
    
  fn main() {
      println!("Hello, world!");
  }
  3. To build the project 
    cargo build
  4. To run the project 
    cargo run
