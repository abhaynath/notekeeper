Using Kotlin Compiler in command-line

  1. Download kotlin-compiler.zip and extract it any folder.
  2. Add path to environment variable in System settings e.g F:\Programs\kotlin-compiler\kotlinc\bin
  3. Create a empty folder 'test'. Create new folder 'src' to hold all source code. Add a new file App.kt in src folder with following code 
    /* 
    test/src/App.kt 
*/
  
  import models.*
  
  fun main() {
      var abhay = Person()
      abhay.show()
  }
  4. Create a new folder 'models' inside src folder. Add a new file Person.kt in models folder with following code 
    
/*
  test/src/models/Person.kt
*/
package models

class Person {
    var firstName: String = "Abhay"
    var lastName: String = "Nath"
    var city: String = "Mumbai"

    fun show() {
        println("Name = ${firstName} ${lastName} \t City = ${city}")
    }
}
  5. To compile the 'src' folder, use kotlinc command. 
    kotlinc src src/App.kt -include-runtime -d test.jar
    This will create a new jar file 'test.jar'
  6. To run a jar file, 
    java -jar test.jar
  7. To compile and run at one command 
    kotlinc src src/App.kt -include-runtime -d test.jar && java -jar test.jar

Hello World Application in Kotlin

  1. Run Intellij Idea. Select 'New Project -> Kotlin'
  2. Type the Project name in the input box.
  3. Intellij will create a 'src' folder.
  4. Right click on src folder, select 'New->Kotlin file/class'. File name is 'app.kt'
  5. Add following code snippet in 'app.kt' 
    
  fun main(){
      println("Hello World Kotlin !")
  }
  6. Build and run application by pressing on green arrow against main function
