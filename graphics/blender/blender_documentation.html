Blender

Build Blender Documentation locally

Make sure Python 3 is installed. To check, run following command

python --version
Create a new folder blender-doc 
cd blender-doc
Clone git repository 
git clone https://projects.blender.org/blender/blender-manual.git
Create a virtual environment 
python -m venv .venv
Change directory to Scripts 
cd .venv/Scripts
Activate the virtual environment 
activate
Install all the required dependencies. You may need to run it from root folder 
pip install -r requirements.txt
Every now and then you may want to make sure your dependencies are up to date using: 
pip install -r requirements.txt --upgrade
To build the pages 
make
