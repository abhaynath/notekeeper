Bake Roughness to image textures

  1. Create a procedural material and apply it on a object.

  2. Create new image texture. Name it something you want.

  3. Add an image texture node in shader editor. Select the image texture you just created.

  4. Select Cycles render engine

    Choose bake type as Roughness

    Select the Mesh Object and the Image Texture node and click on Bake button.

  5. Save the image
