Eevee Freestyle

Render Properties

  1. In 'Render Properties' panel, select Render engine 'Eevee'
  2. Select Render and sampling (Render=20, View=4)
  3. Enable freestyle in Render properties
  4. In Film tab, enable 'Transparent'.

View Layer Properties

  1. Enable 'As Render Pass'.
  2. Select 'Compositing' tab.
  3. Enable 'Use Nodes' in topbar. Connect Freestyle node to image.
