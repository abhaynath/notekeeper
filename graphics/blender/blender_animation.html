Keyframe animation

Perpetual animation loop

  1. Add cube in viewport. Select it and press i. Rotate the cube in z axis by 30 degrees. Press i.
  2. Select the cube and go to graph editor window. Press home key to fit curve in window.
  3. Press a key to select all keyframes and in menubar select linear.
  4. Key → Interpolation method → Linear
  5. Click on the graph line to select it. Go to modifier tab, add modifier cycles in cycles method.
  6. before mode → No cycles
  7. after mode → Repeat with offset
