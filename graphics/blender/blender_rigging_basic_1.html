Blender

Creating mesh

Add 3 seperate cylinder objects in viewport.

Creating Armature

Press Shift + A and add Armature. Extrude root bone to create 4 bones. Last bone is the controller bone.

To make this armature visible on top of mesh, enable In Front in Viewport display.

Controller Bone

Select the Controller bone, and press Alt + p. Select Clear Parent

Inverse Kinematics

Switch to Pose Mode. First select the controller bone, then select the last bone of the armature.

Press Shift + i. Select Add IK - To Active bone.Select the last bone of the armature. You should see the bone constraint added.

Parent mesh to bone

Switch to Object Mode. Select the Cylinder first and then bone.

Now switch to Pose Mode. Select the Cylinder first and then the corresponding bone.

Press Ctrl + p. Select Bone.

Repeat this process for remaining cylinders.

Switch to Pose Mode, select the controller bone and do some keyframe animation.
