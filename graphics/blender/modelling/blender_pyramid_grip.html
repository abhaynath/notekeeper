Pyramid Grip

  1. Add a plane. Subdivide it 20 times horizontally and vertically.

  2. Select all the faces and inset individually twice.

  3. Press Alt + S to shrink flatten

  4. Press M and select Collapse to merge the top vertices.

  5. Add an Empty at the world origin. Make sure Empty and Place are at the same location.

  6. Apply Simple Deform modifier.

  7. Set rotation of Y axis of Empty object.

Diamond Grip

  1. Add a plane. Subdivide it 25 times horizontally and vertically.

  2. Select all faces and poke the faces.

  3. Poked faces will look like this

  4. Now convert these tris to Quads

  5. Deselect all the faces. To select all the tris, click Select faces by sides. Enter 3 in Number of vertices.

  6. To invert the selection, press Ctrl + i

  7. Inset the faces individually twice

  8. Press Alt + s to Shrink flatten

  9. Press M and Collapse.

  10. Add an empty at the same location of plane. Rotate the Empty by 90 degree in y axis. Select the plane and add Simple Deform modifier.
