Revolver - Radial Geometry

  1. Add a plane and make the required changes in the mesh. Apply rotation and scale. Set object origin to Geometry.

  2. Add an empty at the plane origin location. Make sure object origin and empty should be at same location without any rotation and scale.

  3. Add an array modifier and set the count as per requirement.

    Add Simple Deform modifier. Select the Bend option and set empty as origin object. Set angle to 360.

    Rotate the empty in the required axis by 90 degrees.
