Blender

  1. Create a new scene. Add a cube in viewport. Switch to UV Editing tab.

  2. Select Cube and go to edge selection mode. Select the edges you want to mark as seam.

  3. Right click on selected edges and select Mark Seam

  4. Select all edges and press u. Select Unwrap.

  5. In UV menu, select Export UV Layout.

  6. In Export Layout dialogue box, set Fill Opacity to 1.

  7. To use this image texture, we need to create a material. Create a new Material.

  8. In Base Color slot, select Image Texture

  9. Click on open folder, and browse to UV texture.

  10. You can now edit the texture externally in Photoshop or other softwares.

  11. Reload the image texture in 3D Viewport.
