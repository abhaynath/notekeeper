Align Video to main audio

  1. Add the video clip and main audio file in Project bin.
  2. Add video clip anywhere in the timeline Video Track.
  3. Add the main audio file in the timeline Audio Track
  4. Right click on the main audio and select Set Audio Reference. It should take some time to analyze.
  5. Now, right click on the clip and select Align Audio to Reference.
  6. Video clip should be automatically moved to align with reference audio track.
  7. You can either mute the clip audio or delete the clip audio by ungrouping video.
