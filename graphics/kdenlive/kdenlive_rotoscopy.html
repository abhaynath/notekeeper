Rotoscoping

  1. Add the clip in the timeline.
  2. Apply the Rotoscoping effect on the clip
  4. Draw the path around the object you want to mask. Right click to finish the mask.
  6. Use the arrow key to move to next frame.
  7. If the mask is not fit for the object in next frame, add new Keyframe and re-adjust the mask.
  8. Repeat the process for the rest of the clip.
