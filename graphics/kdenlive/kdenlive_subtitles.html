Automatic speech-to-text subtitles

  1. Make sure that speechmodels and other dependencies are downloaded and configured properly.
  2. Click on the audio track you want to generate subtitles for. Make sure it is selected.
  3. Click on the Edit subititle tools. This will enable the subtitle track in the main timeline. Click on the Magic Wand tool.
  4. This should open the Automatic Subtitling Popup. Click on the Process button.
  5. If operation is successful, you should see the generated subtitles in the timeline tracks.
  6. You can export the subtitles file for external editing.
  7. You can customize the subtitle styling as well.
