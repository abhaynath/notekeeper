Krita

Build Krita Documentation

Make sure Python 3 is installed. You can check if it is installed by checking version.

python --version
  1. Create a new folder krita-docs 
    cd krita-docs
    git clone https://invent.kde.org/documentation/docs-krita-org.git
    cd docs-krita-org
  2. set up virtual environment 
    python -m venv PythonEnvTesting
    cd PythonEnvTesting/Scripts/
    activate
  3. install Sphinx and its dependencies. You may need to execute it from root folder. 
    pip install -r requirements.txt
    if you want the autorebuild feature, install sphinx-autorebuild 
    pip install sphinx-autobuild
    start auto-rebuild script 
    sphinx-autobuild . _build/autohtml --open-browser
  4. Build HTML pages 
    make html
    Build EPUB file 
    make epub
