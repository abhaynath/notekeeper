Inkscape

  1. Make sure python3 is installed. Clone the git repository.

    git clone https://gitlab.com/inkscape/inkscape-docs/manuals.git

  2. Change directory to where requirements.txt is located

    cd manuals
    cd Inkscape-Beginners-Guide

  3. Create virual environment.

    python -m venv venv

  4. Activate virual environment.

    venv\Scripts\activate.bat

  5. Install dependencies

    pip install wheel
    pip install -r requirements.txt

  6. Generate HTML pages

    make html
