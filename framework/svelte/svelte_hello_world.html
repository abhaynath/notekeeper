Svelte

Simple hello world application in Svelte

  1. Run following command to create Svelte App 
        npx degit sveltejs/template my-app
    cd my-app
  2. To use TypeScript run following command. 
    node scripts/setupTypeScript.js
  3. Install all packages 
    npm install --verbose
  4. To run the development server, 
    npm run dev
    To change the default port, change 'package.json' scripts.dev settings. 
     "dev": "SET PORT=5474 && rollup -c -w",
  5. To build the project 
    npm run build
