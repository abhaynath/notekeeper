Svelte

Installation using Vite

  1. npm create vite@latest
  2. Select the prefered options
  3. To start development server, 
    npm run dev
  4. To build production release, 
    npm run build

TypeDoc in Vite Svelte

  1. Install typedoc as development dependency. 
    npm install --save-dev typedoc --verbose
  2. Add typedoc.json file in your root directory where package.json is located. 
    {
  "exclude": ["*.svelte"],
  "entryPoints": ["src/main.ts"],
  "out": "docs"
}
    For more details, refer typedoc
  3. Add new command in package.json for documentation. 
    
"scripts": {
  "dev": "vite",
  "build": "vite build",
  "preview": "vite preview",
  "check": "svelte-check --tsconfig ./tsconfig.json",
  "docs": "typedoc --entryPointStrategy expand ./src"
},
  4. To generate the documentation, run following command. 
    npm run docs

Using Props

App.svelte


<script>
import TestFile from "./components/TestFile.svelte";

let players = [];
function playerAdded(val) {
val = val.trim();
if (val.length < 1) {
    return;
}
players.push(val);
players = players;
}
</script>

<div class="main">
  <TestFile title="My Test App" {playerAdded} />
  <h3>Parent</h3>
  <ol>
    {#each players as player (player)}
      <li>{player}</li>
    {/each}
  </ol>
</div>

<style>
  .main {
    padding: 10px;
  }
</style>

TestFile.svelte


<script>
export let title = "";

export let playerAdded;

let playerName = "";

const clickHandler = () => {
playerAdded(playerName);
playerName = "";
};
</script>

<div>
  <div class="tl">{title}</div>
  <input type="text" bind:value={playerName} />
  <button
    on:click={() => {
      clickHandler();
    }}>Send to Parent</button
  >
</div>

<style>
  .tl {
    font-weight: bold;
    font-size: 18px;
  }
