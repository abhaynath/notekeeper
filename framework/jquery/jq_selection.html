Checkbox

To create checkbox, 

    <input type="checkbox" id="chkCountry" checked><label for="chkCountry">Indian</label>
    <div class="output" id="chk_output"></div>
                

    $("#chkCountry").change(function (e) {
        e.stopPropagation();
        var chk_status = $(this).prop("checked");
        if (chk_status) {
            $("#chk_output").html("Checked !");
        } else {
            $("#chk_output").html("Unchecked");
        }


    });

Radio Button Group

To create group of radio buttons, 

    <div>
        <input type="radio" name="framework" value="angular">Angular<br>
        <input type="radio" name="framework" value="react">React<br>
        <input type="radio" name="framework" value="vue">Vue.js<br>
    </div>
Event Handler code for radio group, 

    $("input[type='radio'][name='framework']").change(function (e) {
        e.stopPropagation();
        var val = $("input[name='framework']:checked").val();
        console.log("here : " + val);

    });
