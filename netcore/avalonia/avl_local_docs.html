Setting up local Documentation of Avalonia UI

  1. Go to Avalonia docs repository page
  2. Copy the repository path which will be like 
    https://github.com/AvaloniaUI/avalonia-docs.git
  3. Create a new folder in which you want to set up documentation. Run following command. 
    git clone https://github.com/AvaloniaUI/avalonia-docs.git
    This will download the repository in avalonia-docs folder. 
    cd avalonia-docs
  4. To install all the packages needed, run following command. 
    npm install --verbose
  5. Open docusaurus.config file.

    Change baseUrl value to as per your IIS application name.

    It should look like this, 

    const config = {
    title: 'Avalonia UI',
    tagline: 'Developer Documentation Portal',
    url: 'https://docs.avaloniaui.net',
    baseUrl: '/avalonia/',
    organizationName: 'avaloniaui', // Usually your GitHub org/user name.
    projectName: 'avalonia', // Usually your repo name.
    onBrokenLinks: 'throw',
    onBrokenMarkdownLinks: 'throw',
    i18n: {
      defaultLocale: 'en',
      locales: ['en'],
    },
}

  6. To build the documentation, 
    npm run build
  7. Documentation will be generated in build folder.
